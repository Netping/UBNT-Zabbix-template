Для того, что бы импортировать шаблон, нужно:
Перейдите в: Настройка → Шаблоны
Нажмите на Импорт справа
Выберите импортируемый файл
Оставьте галочки по умолчанию ("Обновить существующее" и "Создать новое" - будут обновлены существующие элементы шаблона, будут добавлены новые элементы, отсутствующие в новом шаблоне удалены не будут)
Нажмите на Импорт
